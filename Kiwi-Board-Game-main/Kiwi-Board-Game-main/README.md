# Kiwi Board Game — Java Implementation

---

## PROJECT OVERVIEW

Kiwi Board Game is a strategic board game implemented in **Java** as part of a software development course project. The game simulates a turn-based board game where players compete by moving pieces strategically across the board.

The objective of the game is to capture territory and outmaneuver the opponent by making intelligent moves based on the game rules. The project focuses on implementing **game logic, board representation, and player interaction** while maintaining a modular and extensible code structure.

The system was designed using **object-oriented programming principles**, allowing easy modification of rules and board configurations.

## GAME RULES

The Kiwi board game follows a turn-based structure where players take turns performing actions on the board.

Basic gameplay mechanics include:

* Moving pieces according to game rules
* Capturing opponent pieces
* Controlling strategic positions on the board
* Winning by achieving the defined victory condition

---

## SYSTEM DESIGN

The project is organized into multiple classes responsible for different aspects of the game.

Example components include:

* **Game Controller** – manages the overall game flow
* **Board** – represents the board layout and piece positions
* **Player** – stores player information and actions
* **Game Pieces** – represent individual movable units

This modular structure improves maintainability and readability.
